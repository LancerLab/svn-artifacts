# Workload Shape Catalog

This document provides a comprehensive overview of all workload shapes and transformations
used in the value numbering framework for shape inference testing across multiple ML frameworks.

## Overview

**Total Coverage**: 17 workload files across multiple operator types
- **Core Operators**: ADD, MATMUL, CONV2D, ReLU, Sigmoid, Softmax, GELU
- **Specialized Operations**: Batch Norm, Layer Norm, Reshape, Transpose, Concat, Split, Max Pool, Embedding, Reduce Mean
- **Subgraph Operations**: Multi-Head Attention

Each operator includes:
- **Static shapes**: 10 test cases with concrete dimensions from real models
- **Dynamic shapes**: 10 test cases with symbolic dimensions and ranges
- **Realistic scenarios**: Shapes derived from BERT, GPT, ResNet, ViT, MobileNet, etc.
